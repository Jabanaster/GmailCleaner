# Production Readiness Checklist

- [ ] **Hosted Backend**: Deploy containerized backend using Render or GCP Cloud Run.
- [ ] **Neon DB Config**: Connect to Neon database with connection pooling enabled.
- [ ] **Secret Rotation**: Rotate token encryption keys and database access credentials.
- [ ] **Security (HTTPS/CORS)**: Enforce HTTPS redirection and restrictive CORS policies.
- [ ] **Rate Limiting**: Apply API route protection and user rate controls to prevent abuse.
